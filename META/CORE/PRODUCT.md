# Noca

> 零摩擦捕获，AI 自动整理，直达 Notion

## 问题

每天遇到各种零散信息（链接、想法、笔记），当下想专注手头的事，不想被打断去整理。结果这些东西要么丢了，要么堆在某个角落。

## 解决方案

1. **快速捕获** - Raycast 快捷键，3 秒完成
2. **AI 整理** - Claude 自动分类格式化
3. **写入 Notion** - 整理好的内容进入你的 inbox

## V1：Raycast 扩展

### 交互流程

```
⌘ + Shift + C（或自定义快捷键）
         ↓
┌─────────────────────────────────────────┐
│  Noca                                   │
├─────────────────────────────────────────┤
│                                         │
│  Content:                               │
│  ┌───────────────────────────────────┐  │
│  │ https://arxiv.org/abs/2401.12345 │  │  ← 自动填入剪贴板
│  └───────────────────────────────────┘  │     可编辑/清空写纯想法
│                                         │
│  Note (optional):                       │
│  ┌───────────────────────────────────┐  │
│  │ 和 forecasting 研究相关           │  │  ← 可选备注
│  └───────────────────────────────────┘  │
│                                         │
│                        ⌘ + Enter: Save  │
└─────────────────────────────────────────┘
         ↓
    存到本地 JSON
         ↓
   "Captured!" 通知
         ↓
    自动关闭，回到你的工作
```

### 使用场景

| 场景 | 操作 |
|------|------|
| 看到有趣链接 | 复制 → `⌘+Shift+C` → 回车 |
| 链接 + 想加备注 | 复制 → `⌘+Shift+C` → Tab → 写备注 → 回车 |
| 纯想法 | `⌘+Shift+C` → 清空第一栏 → 写想法 → 回车 |

### 每日整理

```bash
# 手动触发（V1）
qc-process

# 或者设置定时任务自动跑
```

整理后写入 Notion inbox page。

## V1 范围

### ✅ 包含
- Raycast 扩展
- 读取剪贴板（文字/链接）
- 可选备注
- 纯想法模式（清空剪贴板栏直接写）
- 本地 JSON 存储
- Claude Code 整理脚本
- 写入单个 Notion page

### ❌ 不包含
- 图片支持
- 多 Notion database
- 自动定时整理
- 移动端

---

## 设计原则

1. **3 秒完成** - 快捷键 → 回车 → 结束
2. **零思考** - 不用选分类，AI 来做
3. **不打扰** - 整理在后台，不需要你参与
4. **无额外成本** - 用已有的 Claude subscription
